  ## eslint Rules




**Functions**
 

 - PascalCase *(JSX)*
 - strictCamelCase *(Class Methods)*

 **Variable**
 

 - strictCamelCase
 - UPPER_CASE *(constant values)*

  
**TypeLike**

 - PascalCase
 - Prefix ``I``

**parameter**

 - strictCamelCase
 
 **class**
 - PascalCase
 - Prefix ``I``
